Deduplication of Patent Inventors

Dans le cadre de la migration des données des brevets, dans le système source LOLA, il n’y a pas de table des inventeurs. Les informations sur les inventeurs sont dupliquées, avec de petites erreurs ou de petits changements (n lignes par inventeur). Il faut donc dédupliquer ces informations pour créer une table des inventeurs (1 ligne par inventeur). J’ai mis au point un algorithme de graphes avec un nœud par ligne et des connexions entre les lignes lorsque le nom et le dossier sont identiques. J’ai codé un algorithme simple en python pour calculer les composantes connexes. Une composante connexe correspondait à un inventeur Cet algorithme a été utilisé avec succès pour la migration des données de brevets. Une table avec 16000 inventeurs a été créée. La table a permis de fiabiliser la gestion des inventeurs.
